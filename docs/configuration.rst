.. _brc.py:

``brc.py``
==========

``brc.py`` the configuration that :doc:`b.py` reads from current working directory,
it may look like:

.. code:: python

  service = '<service id>'
  service_options = {
    # see Services section
  }

  handlers = {
    # see Handlers section
  }

  services = {
    # see Services section
  }
